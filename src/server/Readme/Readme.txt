Book A Live Animal Admin UI Application

*** Overview ***

The Book A Live Animal Admin UI is a web-based application developed using Node.js with the Hapi.js framework. 
It provides an interface for users to book live animal inspection slots . 
The app interacts with a backend API for database transactions and leverages caching mechanisms to enhance performance.

*** Key Features ***

Node.js application using the Hapi.js framework for handling server-side logic and routing.

API-driven architecture to interact with the database and perform CRUD operations related to animal booking.

Redis cache integration to store session data and improve performance by reducing database load.

The application is designed with scalability and maintainability in mind, ensuring smooth performance even under high loads.

Asynchronous operations handled with promises and async/await to ensure non-blocking server responses.

Input validation and security features to ensure data integrity and security compliance.


*** Deployment Details ***

Deployment Platform: This application is deployed on the CDP platform for reliable and scalable cloud hosting.
Continuous Integration/Continuous Deployment (CI/CD) pipelines ensure seamless updates and minimize downtime.
Environment Variables are used to configure settings such as database connection details, API keys, and Redis settings.

*** Technologies Used ***

Hapi.js (Node.js Framework): Provides a clean and powerful framework for building scalable web applications.
Redis Cache: For session management and caching frequently accessed data to improve response times and decrease database load.
API Application: Communicates with the database through an external API, ensuring a decoupled and modular system.
MongoDB: For storing persistent data related to users, animals, bookings, etc.
